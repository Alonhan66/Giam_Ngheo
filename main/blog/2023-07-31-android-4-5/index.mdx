## Free account (Backup)

For this release we added _OsmAnd Start_. It means that each our user can register own email for [OsmAnd Cloud](https://osmand.net/docs/user/personal/osmand-cloud).


Now you can make free backup for Favorites and OsmAnd Settings
Free OsmAnd Cloud account for Favorites: _OsmAnd Start_

## Terrain maps updates

We added _Terrain map_ for countries in [Download maps](https://osmand.net/docs/user/start-with/download-maps) menu. This type of data can be used for generated [Slopes, Hillshades(paid)](https://osmand.net/docs/user/purchases/android#free-and-paid-features), and 3D maps([OsmAnd Pro](https://osmand.net/docs/user/purchases/android#pro-features)).

Our updates:
- [Slope](https://osmand.net/docs/user/plugins/contour-lines#slope-map) and [Hillshades](https://osmand.net/docs/user/plugins/contour-lines#hillshade-map) can be generated from _Terrain map_ data

![terrain](./img/terrain.png) ![terrain](./img/terrain_0.png)

- Redesign [_Terrain menu_](https://osmand.net/docs/user/map/configure-map-menu)

![terrain](./img/terrain_1.png) ![terrain](./img/terrain_2.png)

## Android Auto improvments

- New start menu for access Favorites, POIs, Map markers, Tracks, History categories.

![AA](./img/android_auto.png)

   - If current profile is not "Driving", then it should change first in the list Driving.

- Button to hide Screen menu.

## Train profile

_Train_ profile was added for [Profile list](https://osmand.net/docs/user/personal/profiles). New profile allows to calculate railways routes.

![Train](./img/tr.png)

## Contour lines for Map sources

[Contour lines](https://osmand.net/docs/user/plugins/contour-lines#contour-lines-map) works with any ["Map source"](https://osmand.net/docs/user/map/configure-map-menu):

_OsmAnd menu → Configure map → Map source..._

![contour lines](./img/contour_lines.png)

## More actions for external keyboards

Added the expanded list of default actions for [external keyboards](https://osmand.net/docs/user/map/interact-with-map#external-input-device-buttons).

## Announce deviation from the route

Added option to disable / enable "Announce deviation from the route" for [Voice prompts](https://osmand.net/docs/user/navigation/guidance/navigation-settings#voice-prompts) settings:

_OsmAnd menu → Configure profile → Navigation settings → Voice prompts → Other → Announce deviation from the route_

It is enabled by default.

## Update for Start screen

[Start screen](https://osmand.net/docs/user/start-with/first-steps#how-to-download-maps): added option restore OsmAnd state from file.

![Start screen](./img/android_st.png)

## Hazardous Materials classes for North America

Added [Hazardous Materials](https://osmand.net/docs/user/navigation/routing/car-based-routing/#route-parameters---truck) classes for North America:

_OsmAnd menu → Configure profile (Truck) → Navigation settings → Route parameters → Transporting of hazardous materials_

![hazar materials](./img/hazard.png)


## "Via ferrata" for pedestrian routing

Added support of "Via ferrata" for [pedestrian routing](https://osmand.net/docs/user/navigation/routing/pedestrian-routing):

_OsmAnd menu → Configure profile (Walking) → Navigation settings → Route parameters → Montainside route (Via ferrata)_

![Pedestrian route](./img/ferrata.png)

##  "CAI" scale

Added ["CAI"(Club Alpino Italiano)](https://www.cai.it/) scale for [difficulty classification](https://montanarilife.com/2016/07/03/hiking-trail-rating-systems-italy/) of the routes:

_OsmAnd menu → Configure map → Routes → Difficulty classification_

![CAI](./img/cai.png)

## Support themed icons for Android 13

We support themed (adaptive) icons for [Android 13](https://developer.android.com/develop/ui/views/launch/icon_design_adaptive#user-theming).

![Adaptive icon](./img/adaptive_icon.png)

## Option Delete Cloud account

We added to opportunity to delete an account for [OsmAnd Cloud](https://osmand.net/docs/user/personal/osmand-cloud):

_OsmAnd Menu → Settings → OsmAnd Cloud →  → Delete account_

![OsmAnd Cloud](./img/delete_account.png)

## Accessibility updates

[Accessibility](https://osmand.net/docs/user/plugins/accessibility): touch target size for all buttons changed to 48 dp fixed issue with map UI that was visible for screen readers.

## Custom routing files options

[Custom routing files](https://osmand.net/docs/user/navigation/routing/osmand-routing#customize-offline-routing): added options to delete or overwrite.

_OsmAnd menu → Configure profile → Navigation settings → [Navigation type](https://osmand.net/docs/user/navigation/guidance/navigation-settings#navigation-type) → &#8942; (routing_custom.xml) → Delete_

![Delere custom routing](./img/custom_routing.png)
