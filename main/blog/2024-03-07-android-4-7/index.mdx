Good news!  

OsmAnd version 4.7 for Android devices is now available on Google Play and other markets. Read about the new features in this article. 

<!--truncate-->


## What's new

- Current speed / speed limit widget (including Android Auto). 
- Round up numbers in navigation widgets. 
- Allow Custom Quick Action Buttons.
- Ask to disable battery optimization.
- Redesigned graph.
- Apply sorting order to all subfolders – Tracks.
- Filter by Sensor data (HR, Cadence, Power).
- Add support for tags: author, cmt, link, keywords, copyright, email.
- New UI for Open Track screen – Tracks.
- Move to OAuth 2.0 for OpenStreetMap login.
- Ask to disable battery optimization.
- Move sensors tags to gpxtpx:TrackPointExtensio.

## Speedometer

Added the new special widget named Speedometer:

_Menu → Configure screen → Other → Speedometer_

Here you can choose height (_small, middle, big_) for Speedometer and when show Speed limit warning (_Always, When exceeded_).

![Speedometer](./img/speedometer_android.png) ![Speedometer](./img/speedometer_2_android.png)

### Speedometer for Android Auto

Speedometer was added for Android Auto too.

![Speedometer](./img/speedometer_3_android.png)

## Round up numbers

Choose how distance information is displayed in navigation widgets (distance to point, next turn, lanes). Choose _"Precise"_ for exact numbers, or _"Round up"_ for easier reability:

_Menu → Settings → Navigation Profile → General settings → Units & formats → Distance during navigation → Precise_ or _Round up_

![Round up Android](./img/roundup_android.png) ![Round up Android](./img/roundup_2_android.png)

## Custom Buttons

For this release we added Custom Buttons as special form of [Quick action](https://osmand.net/docs/user/widgets/quick-action) tool. It's mean than an user can create a custom action button for device screen.

_Menu → Configure screen → Buttons → Custom buttons → + → Add button_

![Custom button Android](./img/custom_button_android.png) 
![Custom button Android](./img/custom_button_2_android.png)

Next, you can add action or actions for created custom button. And enabled it.

![Custom button Android](./img/custom_button_3_android.png) 
![Custom button Android](./img/custom_button_4_android.png)

Now, you can see the new custom button on the device screen. This button switches on/off the chosen action immediately.

![Custom button Android](./img/custom_button_5_android.png)
