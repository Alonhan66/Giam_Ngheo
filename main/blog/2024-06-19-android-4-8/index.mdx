Good news!  

We're thrilled to announce the release of OsmAnd version 4.8 for Android devices, now available for download on Google Play. This latest update brings a suite of new features and improvements designed to enhance your navigation experience.

**[Get the Update Today!](https://play.google.com/store/apps/dev?id=8483587772816822023)**

To enjoy these new features and improvements, head to Google Play or your preferred app market to download or update to OsmAnd version 4.8. We're committed to making OsmAnd the best navigation tool for Android users, and we're excited for you to experience the latest advancements.

Thank you for your continued support of OsmAnd. Happy navigating!


<!--truncate-->


## What's new

- Visualize elevation changes with [new terrain colorization](#new-terrain-colorization) option "Altitude" 
- Design your perfect map with support for custom color palettes for terrain, tracks, and routes. 
- Improved location position icons with 3D variants 
- New ["Add action" screen](#update-of-quick-actions-tool) of Quick Actions tool
- Quick Actions on [external keyboard](#quick-actions-on-external-keyboard): Assign actions for a personalized experience 
- Weather UI refresh: Wind animation & improved label visibility 
- [OSM Editing](#osm-login): deleted login / password authorization method 
- Switching to system share sheet starting Android 14 
- Fixed track stats, sorting & search 
- Added [POI "Water" category](#category-water)
- Improvements for [3D trak view](#3d-track-improvements)
- GPX track activity



- Buttons Control
- Quick Action Mapping
- Track Activity / Keywords
- My Location 3D
- 3D Track (Finalize)
- Garmin Connect
- Weather animation
- Significant speed improvement for ["Attach to roads"](https://osmand.net/docs/user/navigation/setup/gpx-navigation#attach-to-roads) algorithm

Possible one / many of:

* Traffic Maps

### 3D track improvements

For this release we added the next improvements for [appearance of 3D track](https://osmand.net/docs/user/map/tracks-on-map/track-appearance#3d-track).

- Visualized by: If Visualized by = Fixed height, change Vertical exaggeration to Wall height.

- Wall color: Solid / Downward gradient / Upward gradient, Altitude / Slope / Speed
