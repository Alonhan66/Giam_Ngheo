Hello everybody!

For the last OsmAnd Android release 4.5.6 we added OsmAnd Start plan with Free OsmAnd cloud. 

What is mean?

- Free for everybody.
- Free favorites, settings, OSM edits/notes cloud backup.
- Available for OsmAnd Android version 4.5.6 (soon for OsmAnd iOS version).

   &#9755;  Let's go!

<!--truncate-->

## How to create account?

If you didn't register [your account](https://osmand.net/docs/user/personal/osmand-cloud#login) with [OsmAnd Pro](https://osmand.net/docs/user/purchases/android#pro-features) subscription You can register an email for the [OsmAnd Cloud](https://osmand.net/docs/user/personal/osmand-cloud) and then activate OsmAnd Start.

1. Go to: _OsmAnd App Menu → Settings → OsmAnd Cloud → Create new account_

![Start](./img/start_1a.png)

2. Enter your e-mail address and tap on "_Continue_" button:

![Start](./img/start1.png)

3. Receive a verification code to your e-mail:

![Start](./img/start2.png)

4. Input this verification code (one-time password) and tap on "_Continue_" button:

![Start](./img/start3.png)

5. Opened "OsmAnd Cloud: menu:

![Start](./img/start4.png)


After that you can check OsmAnd Purchases menu:

_OsmAnd App Menu → Settings → Purchases_

Here you can see _**OsmAnd Start**_ package.

|Purchases menu| OsmAnd Start screen|
|------|-------|
|![start](./img/start6.png)|![start](./img/start5.png)|

## How to make a first back up?

With _OsmAnd Start_ you can back up next data: [Favorites](https://osmand.net/docs/user/personal/favorites), [OsmAnd Settings](https://osmand.net/docs/user/personal/global-settings), [Profiles](https://osmand.net/docs/user/personal/profiles), [OSM edits/notes](https://osmand.net/docs/user/plugins/osm-editing#create--modify-poi).

_OsmAnd App Menu → Settings → OsmAnd Cloud → ⚙️ (Back up settings) → Storage taken (Back up data)_
