Hello everybody!

For the last OsmAnd Android release 4.5.6 we added OsmAnd Start plan with Free OsmAnd cloud. 

What does it mean?

- Free for everybody.
- Free favorites, settings, OSM edits/notes cloud backup (_<u>Cloud storage 5 MB</u>_).
- Available for OsmAnd Android version 4.5.6 (soon for OsmAnd iOS version).

   &#9755;  Let's go!

<!--truncate-->

## How to create an account?

If you didn't register [your account](https://osmand.net/docs/user/personal/osmand-cloud#login) with [OsmAnd Pro](https://osmand.net/docs/user/purchases/android#pro-features) subscription You can register an email for the [OsmAnd Cloud](https://osmand.net/docs/user/personal/osmand-cloud) and then activate OsmAnd Start.

1. Go to: _OsmAnd App Menu → Settings → OsmAnd Cloud → Create new account_

![Start](./img/start_1a.png)

2. Enter your e-mail address and tap on the "_Continue_" button:

![Start](./img/start1.png)

3. Receive a verification code in your e-mail:

![Start](./img/start2.png)

4. Input this verification code (one-time password) and tap on "_Continue_" button:

![Start](./img/start3.png)

5. Opened the "OsmAnd Cloud" menu:

![Start](./img/start4.png)


After that, you can check the OsmAnd Purchases menu:

_OsmAnd App Menu → Settings → Purchases_

Here you can see _**OsmAnd Start**_ package.

|Purchases menu| OsmAnd Start screen|
|------|-------|
|![start](./img/start6.png)|![start](./img/start5.png)|

## How to make a first back up?

With _OsmAnd Start_ you can back up the next data: [Favorites](https://osmand.net/docs/user/personal/favorites), [OsmAnd Settings](https://osmand.net/docs/user/personal/global-settings), [Profiles](https://osmand.net/docs/user/personal/profiles), [OSM edits/notes](https://osmand.net/docs/user/plugins/osm-editing#create--modify-poi).

Go to _OsmAnd App Menu → Settings → OsmAnd Cloud → ⚙️ (Back up settings) → Storage taken (Back up data)_ where you need to select the data that will be saved in the backup.

|Backup settings| Back up data |
|------|-------|
|![Backup](./img/backup1.png)|![Backup](./img/backup.png)|

Next click to _Sync now_: _OsmAnd App Menu → Settings → OsmAnd Cloud_ for starting chosen data synchronization. 

| Before|After |
|------|-------|
|![Sync](./img/sync.png)|![Sync](./img/sync1.png)|

Read more about OsmAnd cloud on [our documentation](https://osmand.net/docs/user/personal/osmand-cloud).

## OsmAnd Web

On our website, you can log in with your Email to work with your data.

Go to [www.osmand.net/map](https://osmand.net/map)

1. Click to _Login_ to register:

![WEB](./img/web.png)

2. Enter your Email (OsmAnd Start login) and choose _"Don't have the password or forgot it"_:

![WEB](./img/web1.png)

3. Input your _"New password"_ and code from Email:

![WEB](./img/web2.png)

4. Now, you see your Cloud data on the Web:

![WEB](./img/web3.png)

On our Web service, you can create and add new Favorite points for your account.

5. Click on your _Login name_ (Left top corner) you can see info about your account, download backup, change your e-mail or delete your account:

![WEB](./img/web4.png)

P.S. With [OsmAnd Pro](https://osmand.net/docs/user/purchases/android#prices) users can have more features with [OsmAnd Cloud](https://osmand.net/docs/user/personal/osmand-cloud): back up all OsmAnd data, create on the web and sync GPX tracks.
