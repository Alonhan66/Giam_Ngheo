Hello Everyone,

We're delighted to share some exciting news with you! OsmAnd version 4.8 for iOS devices is now officially available. This latest update brings a host of new features and improvements that we can't wait for you to try.

You can download the update right now from the [AppStore](https://itunes.apple.com/us/app/osmand-maps-travel-navigate/id934850257). Added new widgets, improved user interface, and several performance upgrades that make OsmAnd even better.


<!--truncate-->

## What's new

• New "Speedometer" widget compatible with CarPlay
• Added Turn-by-turn instructions to route details
• Configure the map screen by adding multiple "Quick Action" buttons
• 3D Track Improvements: new visualization and colorization options
• Added improved multi-selection mode for Tracks
• Fixed names for bearing widgets
• Add smooth transitions while zooming in 3D mode
• Fixed "Radius ruler" visibility in all widget panels
• "Location position on screen" is now in Configure Screen
• Change "Wheel circumference" in BLE sensor settings
• Added separate "Speed limit" navigation alert
