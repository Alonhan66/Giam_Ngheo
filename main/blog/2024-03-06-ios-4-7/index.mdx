Hello, everyone!

We are pleased to announce the release of a 4.7 version of OsmAnd for iOS devices. [Update it at the AppStore](https://itunes.apple.com/us/app/osmand-maps-travel-navigate/id934850257).

<!--truncate-->

## What's new

- Enhanced top and bottom widget panels to accommodate all widgets.
- Improved management for tracks visible on the map.
- Updated "Analyze on Map" with the ability to display metrics from sensors.
- Enhanced tags support for GPX has been added: link, author, copyright, and keywords. 
- Improved widget configuration by allowing users to move pages and rows.
- Fixed the "Coordinates" widget where, in some cases, the last digits were missed.
- Fixed "3D Mode" button behavior: tapping on the button doesn't save the tilt angle.
- Fixed status bar color for older iOS versions.
- Use App Language for Basemap.
- New "Sun position" widget.
- OsmAnd Cloud: added ability to restore deleted files from trash.
- OsmAnd Cloud: added ability to delete account.
- Mac OS version: fixed rotate and pinch to zoom gestures.
- Add support for additional tags for OSM routes.
- "My Places > Tracks" redesign.


## Redesign My Places > Tracks menu

We made full redesign [_Tracks menu_](https://osmand.net/docs/user/personal/tracks#my-places-menu) of [_My Places_](https://osmand.net/docs/user/personal/myplaces). 

Added new views of:  
- Track folders,
- Track list, 
- _Tracks on map_ menu,
- _New track_ - recording new track. Button for Start track recording.

![My Places > Tracks menu iOS](./img/my_places_tracks_menu_ios.png) 

_Tracks on map_ folder opens the [_Tracks menu_](#redesign-tracks-menu-of-configure-map) of Configure map.

By clicking to _Three points_ button you open the menu with the next actions:
- _Add folder_ - creation and name the new folder.
- _Import_ - import action of GPX file.

![My Places > Tracks menu iOS](./img/my_places_tracks_menu_2_ios.png) 


_New track_ allows to start [_Trip recording_](https://osmand.net/docs/user/plugins/trip-recording) action. Just click _"Start recording"_ button. During recording you can see Recording track data: distance, duration, number of waypoints. There are two points during recording: _Stop record_ and _Save GPX track_.

![My Places > Tracks menu iOS](./img/my_places_tracks_menu_3_ios.png) ![My Places > Tracks menu iOS](./img/my_places_tracks_menu_4_ios.png)

The long tap to any folder opens _Folder_ menu with the next actions:
- _Rename_ - allows rename a selected folder.
- _Appearance_ - opens the [_Appearance menu](https://osmand.net/docs/user/map/tracks-on-map#track-appearance) of tracks for a selected folder.
- _Export_ - allows [to export folder as OSF-file](https://osmand.net/docs/user/personal/import-export#export).
- _Move_ - allows to move a folder to another Tracks folder or to add the new folder for moving selected folder to it.
- _Delete_ - allows to delete a selected folder.

![My Places > Tracks menu iOS](./img/my_places_tracks_menu_5_ios.png)


The long tap to any GPX track opens the menu with the next actions:
- _Show on map_ - shows a selected track on the map.
- _Appearance_ - opens track [_Appearance menu](https://osmand.net/docs/user/map/tracks-on-map#track-appearance).
- _Navigation_ - [navigate by track](https://osmand.net/docs/user/navigation/setup/gpx-navigation).
- _Analyze_ - opens [_Analyse on map_](https://osmand.net/docs/user/navigation/setup/route-details#analyse-on-map).
- _Share_ - shares a selected track.
- _Upload modifications to OSM_
- _Edit_ - opens a selected track on [_Plan a route_](https://osmand.net/docs/user/plan-route/create-route).
- _Duplicate_ - allows to duplicate a selected track.
- _Rename_ - allows to renamve a selected track.
- _Move_ - allows to move a selected track to chosen folder.
- _Delete_ - allows to delete a selected track.

![My Places > Tracks menu iOS](./img/my_places_tracks_menu_6_ios.png)

## Sun position widget

New [_Sun position_ widget](https://osmand.net/docs/user/widgets/info-widgets#sun-position) was added. It's additional to Sunrise and Sunset widgets. Now this section of widgets names _"Sun position"_:

_Menu ‚Üí Configure screen ‚Üí Widgets ‚Üí Add widget ‚Üí Sun Position ‚Üí Sun position, Sunrise, Sunset_. 

![Sun Position](./img/sun_position_3_ios.png)

_"Sun Position"_ widget shows the time until sunrise or sunset, depending on the current time. This widget has the next settings:
- _Mode_ - choose _"Next event"_, _"Sunset"_, _"Sunrise"_
- _Format_ - here you choose _"Time left"_ or _"Next event"_, this format settings can be modified by clicking to the widget on the screen.

![Sun Position](./img/sun_position_ios.png) ![Sun Position](./img/sun_position_2_ios.png)
