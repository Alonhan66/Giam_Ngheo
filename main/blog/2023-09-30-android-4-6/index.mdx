Good news, everybody!  
Version 4.6 of the OsmAnd program for Android devices has undergone revolutionary changes. Go and read what has been added and improved.  


<!--truncate-->


- Updates for widgets:
   - Updates for Top/Bottom panels in the Configure screen menu.
   - Added Glide Ratio Widget.
   - Added Temperature widget for External sensors.
- New Local tab in Maps & Resources menu.
- New feature Smart Folder in Tracks menu.
- Filters for Tracks search.
- Added Custom Quick Action buttons.
- Adding a screen Trash to view and manage files deleted from OsmAnd Cloud.
- New Help screen with offline support.
- Redesigned Weather screen.
- Improved memory management for large GPX tracks.

### Added Glide Ratio Widgets

![Glide ratio](./img/glide_ratio.png)

Simple _[Glide Ration](https://en.wikipedia.org/wiki/Lift-to-drag_ratio#Glide_ratio)_ widgets were added for Aircraft usage. Widgets show the glide ration to the target or avearge glide for the current trip:

 _OsmAnd menu ‚Üí Configure screen ‚Üí Widgets (Left / Right panel) ‚Üí [Glide ratio to target](https://osmand.net/docs/user/widgets/info-widgets/#glide-ratio)_ and [_Average glide ratio_](https://osmand.net/docs/user/widgets/info-widgets/#glide-ratio)

- _Glide ratio to target_ shows the glide ration needed to reach the target point.
- _Average glide ratio_ shows the average glide ratio for the set interval. For this widget you can specify the time interval for glide averaging.

|Glide ration widgets| Widgets on the screen|

Next you need [to add the temperature widget](https://osmand.net/docs/user/widgets/configure-screen/#right--left-panel-widgets) to the OsmAnd screen:

_Menu ‚Üí Configure screen ‚Üí [Left panel /Right panel](https://osmand.net/docs/user/widgets/configure-screen/#right--left-panel-widgets) ‚Üí External sensors ‚Üí Temperature_
